zsh で調べると Oh My Zsh の記事を多く見かけるので入れてみることにした  
インストール用のシェルを実行するだけで OK  

```sh
sh -c "$(curl -fsSL https://raw.github.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"
```

プロンプトや補完がわかりやすくなる
